# Mighty Doodle

Welcome to the open-source ecosystem of Mighty Doodle — a playful literacy platform designed to help children build reading and writing skills through interactive, game-based learning.

## 🧩 Repositories

- 🎮 **Unity App**  
  https://github.com/Mighty-Doodle/mighty-doodle-app  
  The main application where children learn through engaging mini-games.

- ⚙️ **App Service (Backend for App)**  
  https://github.com/Mighty-Doodle/mighty-doodle-appservice  
  Serves the Unity app and handles core backend functionality.

- 🌐 **Web Service (Backend for Portal)**  
  https://github.com/Mighty-Doodle/mighty-doodle-web-service  
  Powers the parent/teacher portal.

- 🖥️ **Web Admin (Portal UI)**  
  https://github.com/Mighty-Doodle/mighty-doodle-web-admin  
  Frontend for the parent/teacher dashboard.

---

## 🚀 Getting Started

Each repository includes detailed setup instructions. To run the full system:

1. **Set up the database**  
2. **Run the App Service (backend for Unity app)**  
3. **Run the Web Service (backend for portal)**  
4. **Start the Web Admin (portal UI)**  
5. **Open and run the Unity app**
